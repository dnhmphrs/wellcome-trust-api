# wellcome-rust-api
API for Wellcome Trust Code Challenge
- Derived from Example: https://github.com/actix/examples/tree/master/https-tls/awc-https
- Cargo is required to compile locally: https://www.rust-lang.org/tools/install

###  Build

```
$ cargo build
```

###  Serve

```
$ cargo run
```

###  Clean

```
$ cargo clean
```
