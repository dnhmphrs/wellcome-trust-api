# wellcome-trust-api
API for Wellcome Trust Code Challenge

##  /python - Python Build
- Rebuild in Python after Rust hurt my soul
- Run Locally - available at localhost:5000

###  Run Locally
```
$ cd /src/python
$ flask run
```

- Cargo is required to compile locally: https://www.rust-lang.org/tools/install

###  Cargo Commands

```
$ cargo build
$ cargo run
$ cargo clean
```
